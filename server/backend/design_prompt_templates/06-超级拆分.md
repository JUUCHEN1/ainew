# 超级拆分

来源：`backend/app/api/endpoints/comic_prompts.py`，`ASSET_DESIGN_TEMPLATES` 中 `id="super-split"`。

```text
根据上传的主体形象，生成一张「全景式角色深度概念分解图」。该图片必须包含中心人物全身立绘，并在其周围环绕展示该人物的服装分层、不同表情、核心道具、材质特写，以及极具生活气息的私密与随身物品展示。
构图布局:
• 中心位: 放置角色的全身立绘或主要动态姿势
• 环绕位: 在中心人物四周空白处，有序排列拆解后的元素
• 视觉引导: 使用手绘箭头或引导线连接
拆解内容:
服装分层: 将角色的服装拆分为单品展示
表情集: 在角落绘制 3-4 个不同的头部特写
材质特写: 选取 1-2 个关键部位进行放大特写
画风: 保持高质量的 2D 插画风格，线条干净利落
背景: 使用白色背景
文字说明: 在每个拆解元素旁模拟手写注释
```
