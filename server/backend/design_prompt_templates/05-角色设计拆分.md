# 角色设计拆分

来源：`backend/app/api/endpoints/comic_prompts.py`，`ASSET_DESIGN_TEMPLATES` 中 `id="design-split"`。

```text
# Role (角色设定) 你是一位顶尖的游戏原画设计师。请以我上传的这张图片为唯一视觉基准，为其创作一张专业级的「角色三视图及细节设定图」。

# Style Inheritance (风格继承) 禁止修改画风。请完全复制原图的渲染材质、色彩饱和度、线条风格以及光影逻辑。生成的图像必须看起来像是同一位画师在同一时间创作的作品。

# Layout & Composition (空间布局指令) 请将画布划分为以下四个功能区：

【中央主视区】：展示角色的全身三视图（正面、90度正侧面、背面）。角色需呈平视视角站立，动作自然统一，确保服装的结构在视角转换时逻辑自洽（例如：正面看到的佩囊，在侧面和背面应出现在对应的位置）。

【左侧特写区】：在画面左上方，提供一张该角色的高精细度头像（Headshot），重点展示其五官特征、眼神神态以及发饰细节。

【底部细节区】：在画面底部，水平排列3-4个特写圆框或方框。分别提取并放大展示：服装的材质纹理、腰间的饰品细节、手部的武器或手套细节。

【右侧信息区】：在最右侧边缘，绘制一条带有厘米数字的垂直身高刻度尺，并在其旁边垂直排列5-8个正方形色块，展示该角色设计的主要配色方案（Color Palette）。

# Consistency Rules (一致性控制)
面部一致：不同视角下的五官比例、瞳色、妆造必须完全相同。
服装一致：所有视角的服装层次、褶皱逻辑和配饰数量必须保持严格同步。
背景规范：使用极简的纯色背景（如低饱和度的浅灰或米色），确保没有任何杂色干扰。

# Technical Quality (技术要求) 极致细节，超高分辨率，构图对齐精准，光影统一，整体呈现出官方艺术设定集的质感
```
