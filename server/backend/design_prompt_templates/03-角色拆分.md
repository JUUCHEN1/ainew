# 角色拆分

来源：`backend/app/api/endpoints/comic_prompts.py`，`ASSET_DESIGN_TEMPLATES` 中 `id="character-split"`。

```text
横图，白色背景，创作例图的角色的设计图。
要求：
1、 不要照搬原图
2、分解构造各种细节（用局部放大图的方式呈现，用中文进行标注）
3、保持好角色本体的现有特征，例如脸型、发色、身材等归属于人体特征的内容
4、采用图片里的绘画风格不变 。
整张设计图被清晰地划分为五个主要区域：
1、左上区域：三视图展示，包括正视图、侧视图、背视图
2、左下区域：拆分角色身上衣服，每件衣服进行单独特写
3、中上区域：角色的饰品独立拆分
4、中下区域：不同的表情拆分，包括开心、惊讶、生气、害羞等
5、右侧区域：角色站立的全身图。
```
